# Discord Onboarding — мапінг опцій на ролі

Налаштовується вручну: Server Settings → **Onboarding** (Discord API не дає
створювати onboarding-питання ботом). Ролі вже створені `pnpm discord:apply`
(`infra/discord/config/roles.ts`, ADR-023).

> Onboarding вимагає Community-фічі та принаймні 7 каналів у Default Channels.
> Якщо wizard скаржиться — спершу завершіть apply (19 каналів вистачає).

## Default Channels (видимі всім новим до проходження питань)

- #правила
- #оголошення
- #welcome
- #загальний-чат
- #показуй-проєкти
- #поради-з-проєктування
- #ресурси

## Питання 1 — «Хто ви?» (multi-select OFF, обов'язкове)

| Опція (emoji + текст)           | Видає роль             |
| ------------------------------- | ---------------------- |
| 💡 DIY-ентузіаст                | `💡 i-am/DIY`          |
| 🏢 Малий бізнес (меблі, HoReCa) | `🏢 i-am/Business`     |
| 🛠️ Інженер / конструктор        | `🛠️ i-am/Engineer`     |
| 🏭 Виробник (різання/гибка)     | `🏭 i-am/Manufacturer` |

## Питання 2 — «Що вас цікавить?» (multi-select ON, опціональне)

| Опція                                      | Видає роль               | Відкриває категорію |
| ------------------------------------------ | ------------------------ | ------------------- |
| 🐛 Повідомляти про баги і питати підтримку | `🐛 interest/Support`    | 🐛 ТЕХ-ПІДТРИМКА    |
| 🎨 Поради з проєктування                   | `🎨 interest/Design`     | — (відкрита)        |
| 🏭 Вибір виробника, k-фактор, допуски      | `🏭 interest/Production` | 🏭 ВИРОБНИЦТВО      |
| 📚 Навчання з нуля                         | `📚 interest/Learning`   | 📚 НАВЧАННЯ         |

## Після збереження wizard'а — перевірка

1. Інкогніто-акаунтом зайти за invite-лінком, пройти onboarding.
2. Вибрати лише «📚 Навчання з нуля» → видно НАВЧАННЯ, НЕ видно
   ТЕХ-ПІДТРИМКА і ВИРОБНИЦТВО.
3. `pnpm discord:diff` → onboarding не створює drift (ролі ті самі).

> Authority-ролі (Founder/Moderator/Verified-Manufacturer/Member) в onboarding
> НЕ беруть участі: Member видається автоматично (Onboarding → Default Role
> або вручну), Verified-Manufacturer — вручну після верифікації виробника.
