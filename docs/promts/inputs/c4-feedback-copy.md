# C4 — UX-тексти форми виробничого фідбеку `/f/{export_id}`

> **Джерело:** промпт C4 з `docs/15_LLM_PROMPTS.md`. Тон: дружній до майстра у цеху, без канцеляриту, коротко (людина стоїть біля верстата).

Два варіанти на вибір. **Обраний = Варіант 1** (коротший і тепліший, ★). Варіант 2 — у Опитуванні як альтернатива.

## Варіант 1 (★ обрано)

### UA

| Секція                 | Текст                                                                                                  |
| ---------------------- | ------------------------------------------------------------------------------------------------------ |
| **H1** (заголовок)     | Як вийшла деталь?                                                                                      |
| Поле 1 · «виготовлено» | **Деталь вийшла?** • ✅ Так, все ок / ⚠️ З відхиленнями / ❌ Не вийшла                                 |
| Поле 2 · «відхилення»  | **Розмір десь не збігся? Напиши, де і на скільки мм**<br/>_(placeholder: наприклад: «полиця +0.3 мм»)_ |
| Поле 3 · «коментар»    | **Що сказати нам?** _(placeholder: обов'язково — якщо не вийшла; необов'язково — інакше)_              |
| Кнопка                 | Надіслати                                                                                              |
| Toast · подяка         | Дякуємо! Це допомагає нам покращувати креслення.                                                       |
| Toast · помилка        | Не вдалося надіслати. Спробуй ще раз або напиши нам у Discord.                                         |

### EN

| Section               | Text                                                                                            |
| --------------------- | ----------------------------------------------------------------------------------------------- |
| **H1**                | How did the part come out?                                                                      |
| Field 1 · "made"      | **Did the part come out?** • ✅ Yes, all good / ⚠️ With deviations / ❌ Didn't come out         |
| Field 2 · "deviation" | **Any size mismatch? Tell us where and how many mm**<br/>_(placeholder: e.g. "shelf +0.3 mm")_  |
| Field 3 · "comment"   | **What would you tell us?** _(placeholder: required if it didn't come out; optional otherwise)_ |
| Button                | Send                                                                                            |
| Toast · thanks        | Thanks! This helps us improve the drawings.                                                     |
| Toast · error         | Couldn't send. Try again or ping us on Discord.                                                 |

## Варіант 2 (альтернатива для Опитування)

Довший, більш «формальний» — можливо ближче до звичної UX-мови, але менш «цехово-тепло».

### UA

| Секція          | Текст                                                                                            |
| --------------- | ------------------------------------------------------------------------------------------------ |
| **H1**          | Зворотний зв'язок про виготовлення                                                               |
| Поле 1          | **Статус виготовлення:** Так / Ні / Частково                                                     |
| Поле 2          | **Відхилення від креслення:** розмір і місце (в мм)<br/>_(placeholder: «висота -0.5 мм, знизу»)_ |
| Поле 3          | **Додаткові коментарі:** _(вільна форма)_                                                        |
| Кнопка          | Надіслати відгук                                                                                 |
| Toast · подяка  | Ваш відгук зафіксовано. Дякуємо!                                                                 |
| Toast · помилка | Помилка відправлення. Перевірте з'єднання і повторіть.                                           |

### EN

Аналогічно, ключ: «Feedback about production» / «Production status» / «Deviation from drawing» / «Additional comments» / «Submit feedback».

## Обґрунтування вибору

**Критерії з C4:** «дружній до майстра, без канцеляриту, коротко». Варіант 1 виграє за всіма трьома:

- **Дружній:** «Як вийшла деталь?» замість «Зворотний зв'язок про виготовлення» — розмовна форма.
- **Без канцеляриту:** «Що сказати нам?» замість «Додаткові коментарі:» — питання, не бланк.
- **Коротко:** H1 в 3 слова vs 5 слів; toast «Дякуємо!» vs «Ваш відгук зафіксовано».

Стани поля 1 позначено емоджі (✅ ⚠️ ❌) — швидкий візуальний parsing на телефоні під сонцем біля верстата.

**Питання до yurii:** Варіант 1 (★) чи 2? Дефолт при merge = Варіант 1 (обрано у web-коді).
