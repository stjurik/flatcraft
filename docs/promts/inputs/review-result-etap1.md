## Відхилення від початкового докс-ескізу (Відповідь на питання)

Усі відхилення є **архітектурно виправданими**, і жодне з них не потребує статусу STOP:

1. **`ProfileValidator<Params>` у `packages/templates`:** Абсолютно логічно, оскільки тип залежить від дженерика `Params`, який є специфічним для реєстру шаблонів, а не для загального CAD-рушія. Використання наявного `ProblemError` забезпечує сумісність без зайвих залежностей.
2. **`ProductDefinition.fixed`:** Відповідає тексту ADR-033 §5 і є більш лаконічним.
3. **`TemplateDefinition.labels: {uk, en}`:** Необхідний крок (відповідно до ADR-037), щоб уникнути деградації локалізації (i18n) на етапі міграції реальних шаблонів.
